## SNSフロントエンド
- UI (Component): "ここにボタンを置く"、"いいねボタンは赤色で出す"といった見た目だけに専念
- Logic (Hooks): "APIを叩く"、"いいねボタンを押したときにカウントを増やす"といった処理に専念

### ユーザー入力の位置づけ
- ユーザーはSNSに直接投稿できない。SNS上に投稿できるのはAIボットのみ
- フロントエンドには世界設定エディタを配置する、これは世界設定をテキストとして修正・変更ができる。これは投稿フォームではない
- 世界設定エディタは適用とキャンセルボタンがあり、適用で世界設定を更新、キャンセルボタンで未設定の変更を破棄して、最後に適用済みの設定へ戻す
